# TV Barlow Typography Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the TV board's rounded Nunito face with the squarer Barlow family.

**Architecture:** Keep the 500, 600, and 700 font weights inside the lazy-loaded `TvWorkspace` chunk and scope the family to `.tv-board`. Replace the Fontsource dependency rather than retaining two TV-only font packages.

**Tech Stack:** React, CSS, Fontsource variable fonts, Vitest, Vite

---

### Task 1: Specify the Barlow font contract

**Files:**
- Modify: `src/features/tv/__tests__/TvWorkspace.font.test.js`
- Modify: `src/features/tv/__tests__/tvBoardStyles.test.js`

**Step 1: Write failing expectations**

Expect `TvWorkspace.jsx` to import Barlow's 500, 600, and 700 stylesheets and `.tv-board` to use `'Barlow', system-ui, sans-serif`.

**Step 2: Verify failure**
